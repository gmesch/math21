Features

  • Single HTML file; slides are DIVs
  • Navigation by click on nav indicator and by keyboard cursor keys
  • Browser history and bookmarks work
  • Fully functional maps API application (press 1 any time)
  • Embedded iframes load only on display
  • onshow/onhide event handlers on slides
  • Progressive display
  • Syntax highlighting for code fragments
